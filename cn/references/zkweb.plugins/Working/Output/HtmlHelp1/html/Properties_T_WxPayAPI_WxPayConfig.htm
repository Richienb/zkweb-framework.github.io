WxPayConfig Properties

APP和网页支付提交用户端ip，Native支付填调用微信支付API的机器IP
KEY
商户支付密钥，参考开户邮件设置（必须配置） 例如 e10adc3849ba56abbe56e056f20f883e
MCHID
商户号（必须配置） 例如 1233410002
NOTIFY_URL
支付结果通知回调url，用于商户接收支付结果 例如 http://paysdk.weixin.qq.com/example/ResultNotifyPage.aspx
SSLCERT_PASSWORD
证书密码 例如 1233410002
SSLCERT_PATH
证书路径 应该是APP_Data下的路径 例如 cert/apiclient_cert.p12
