The DefaultOrderChecker type exposes the following members.

Methods

CanConfirm
判断订单是否可以确认收货 前台使用 允许条件: 订单状态是已发货
CanDeliveryGoods
判断订单是否可以发货 后台使用 允许条件: 订单状态是等待发货
CanEditCost
判断订单是否可以修改价格 后台使用 允许条件: 订单状态是等待付款，且关联的收款交易的状态是初始状态
CanEditShippingAddress
判断订单是否可以编辑收货地址 后台使用 允许条件: 订单状态是等待付款，或等待发货
CanPay
判断订单是否可以付款 前台使用 允许条件: 订单状态是等待付款，且有可支付的关联交易
CanProcessAllGoodsShipped
是否可处理订单所有商品已发货 修改订单状态时使用 允许条件: 订单状态是等待发货，且所有商品都已经发货
CanProcessOrderPaid
是否可处理订单已付款 修改订单状态时使用 允许条件: 订单状态是等待付款，且关联的所有交易状态是交易成功或担保交易已付款
CanProcessSuccess
是否可处理交易成功 修改订单状态时使用 允许条件: 订单状态是已发货
CanSetCancelled
判断订单是否可以取消 前台使用 允许条件: 订单状态是等待付款，且关联的收款交易的状态是初始状态或等待付款
CanSetInvalid
判断订单是否可以作废 后台使用 允许条件: 订单状态不是已作废或已取消
