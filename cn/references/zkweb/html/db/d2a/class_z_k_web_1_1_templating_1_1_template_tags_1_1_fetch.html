详细描述

把指定路径的执行结果设置到变量 指定的路径可以是get也可以是post，会自动检测 路径中的变量的获取顺序

  • 等于"*query"时使用当前请求的参数
  • 当前模板上下文中的变量
  • 当前http上下文中的参数 执行结果
  • 结果是JsonResult或PlainResult时使用返回的结果
  • 其他结果时使用描画的内容，但不支持二进制

{% fetch /api/example_info > example_info %} {{ example_info }} url支持变量 {% fetch /api/example_info?id={id} > example_info %} {% fetch /api/example_info?{*query} > example_info %} {{ example_info }}
