详细描述

测试运行器 运行器对应一个指定的程序集 运行测试的流程

  • 在程序集中查找带有UnitTestAttribute属性的类
  • 创建这个类的实例，每个类都会有一个单独的线程
  • 查找所有公开的函数，每个函数都作为一个测试
  • 执行这些函数并向事件处理器报告结果
  • 如果类继承了IDisposable，调用Dispose函数
