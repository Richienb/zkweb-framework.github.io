The IDatabaseContext type exposes the following members.

Methods
  NameDescription
BatchDeleteT
Batch delete entities Return how many entities get deleted It should call registered `IEntityOperationHandler`
BatchSaveT
Batch save entities It should call registered `IEntityOperationHandler` Update action can be used for operation between `BeforeSave` and `AfterSave` callback
BatchUpdateT
Batch update entities Return how many entities get updated It should call registered `IEntityOperationHandler`
BeginTransaction
Begin a transaction It should support recursive calling Only the first call will create a transaction, later calls just increase the counter Attention: Not all ORM support this feature
CountT
Get how many entities that matched the given predicate
DeleteT
Delete entity from database It should call registered `IEntityOperationHandler`
FinishTransaction
Finish the transaction It should support recursive calling Only the last call will commit the transaction, previous calls just decrease the counter Attention: Not all ORM support this feature
GetT
Get single entity that matched the given predicate It should return null if no matched entity found
QueryT
Get the query object for specific entity
RawQueryT
Perform a raw query to database No operation handler will be called `query` and `parameters` are ORM and Database specified
RawUpdate
Perform a raw update to database No operation handler will be called `query` and `parameters` are ORM and Database specified
SaveT
Save entity to database It should call registered `IEntityOperationHandler` Update action can be used for operation between `BeforeSave` and `AfterSave` callback
