详细描述

插件管理器 创建时会载入所有插件 载入插件的流程 枚举配置文件中的Plugins 载入Plugins.json中的插件信息 使用Csscript编译插件目录下的源代码到dll 载入编译好的dll 注册dll中的类型到Ioc中 注意 载入插件后因为需要继续初始化数据库等，所以不会立刻执行IPlugin中的处理 IPlugin中的处理需要在创建这个管理器后手动执行
