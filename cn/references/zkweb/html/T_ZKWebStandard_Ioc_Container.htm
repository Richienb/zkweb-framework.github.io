Container Class
Ioc容器 容器特性 - 不支持构造函数注入，添加的类型必须带无参数的构造函数 - 不支持构造函数注入的原因 - 减少注册时间 - 不需要编写大量多余的代码 性能测试 - Register Transient: 1000000/2.3s (DryIoc: 6.1s) - Register Singleton: 1000000/2.6s (DryIoc: 5.2s) - Resolve Transient: 10000000/3.2s (DryIoc: 0.54s) - Resolve Singleton: 10000000/3.2s (DryIoc: 0.43s) - ResolveMany Transient: 10000000/11.8s (DryIoc: 14.7s) - ResolveMany Singleton: 10000000/11.1s (DryIoc: 12.9s)
